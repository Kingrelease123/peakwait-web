# peakwait-web

The public marketing site for **PeakWait** (PeakWait LLC), served via GitHub Pages at **peakwait.net**.

- `index.html` — landing page
- `privacy/index.html` — Privacy Policy (peakwait.net/privacy)
- `terms/index.html` — Terms of Service (peakwait.net/terms)
- `CNAME` — custom domain binding for GitHub Pages

Static, self-contained HTML/CSS/JS — no build step. Edit a file, commit, and Pages redeploys.

> Legal pages are an engineering draft; have counsel review before relying on them publicly.
